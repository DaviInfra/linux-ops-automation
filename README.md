# 🐧 Linux Ops Automation

Este repositório contém ferramentas de automação para administração de servidores Linux, focadas em **segurança (Hardening)** e **monitoramento proativo** de recursos.

## 🛠️ O que este projeto faz?
- **Health Check:** Varredura rápida de uso de CPU, Memória RAM e Disco.
- **Security Check:** Validação de status de Firewall (UFW) e conectividade.
- **Automação de Diagnóstico:** Gera um relatório rápido no terminal para análise de incidentes por equipes de Suporte N2 e N3.
